# Pandas-Challenge

#### Heroes of PyMoli

This script looks gaming company data for the game Heroes of Pymoli.  It breaks down the data to determine the following:

- Total Number of Unique Players
- Overall purchasing, including total number of purchases, total revenue, and average spending.
- Spending based on Items, Gender, Age Category.
  - Purchase Count
  - Average Purchase Price
  - Total Purchase Value
  - Average Purchase Total
